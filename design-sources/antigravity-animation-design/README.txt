二、核心技术优化说明
------------------------------------------------------------------------
本 V6 交付版本专门针对常规粒子背景的“高延迟”与“卡顿抖动”实施了多项物理优化：

1. [几何透视逆投影 (低交互延迟)]
   - 传统粒子多使用 CPU 端 Raycast 射线网格碰撞求交来获取鼠标的 3D 世界位置，这不仅造成高频
     CPU 性能占用，还常常需要进行“隔帧降频”处理，带来严重的交互滞后感。
   - 本项目通过纯代数公式进行逆向三维透视投影，在每一帧以 0 CPU 延迟的超低输入开销获取 100%
     实时精准的鼠标世界坐标。鼠标在每一帧都能满帧跟手、绝对丝滑。

2. [敏捷物理缓动系数 (0.08)]
   - 将粒子排斥力场中心的追踪插值因子升级为 0.08。相比旧版的 0.02，该设定极大消除了交互时的
     沉重黏滞感，保证排斥力场“跟手性”拉满，快速划过屏幕时粒子可以极速闪避弹开。

3. [纯手动时钟片 dt 计算 (消除时钟抖动)]
   - 彻底移除了 Clock.getDelta()，完全使用 time - lastTime 手工计算时间片，消除了由
     Three.js 时钟内部状态冲突引起的帧抖动与粒子微细颤抖，运动平稳流畅。

4. [粒子保底尺寸 minScale]
   - 修改了 Vertex Shader 的 gl_PointSize 逻辑，即使在无鼠标划过的闲置区域，粒子也有基础
     保底大小，保证大面积粒子背景隐约可见，科技感拉满。

三、如何移植与集成到新项目
------------------------------------------------------------------------
如果您想将此 WebGL 背景集成到您自己的网站：
1. 引入 Three.js 库 (例如 v0.128.0)。
2. 在您的 HTML 中放置一个全屏 Canvas：`<canvas id="antigravityCanvas"></canvas>`。
3. 复制 `index.html` 中从 `class JA` 起到 `initWebGLAntigravityParticles()` 的 JS 逻辑，
   配置您的场景即可直接运行。
